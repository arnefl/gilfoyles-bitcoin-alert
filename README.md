## Prerequisites
* Windows, Mac OS X
* Python 3, with the following libraries
  * winsound (Windows)
  * pyaudio and wave (Mac OS X)
  * urllib3
  * tempfile
  * json
  * time
  * youtube_dl
  * subprocess
  * (os, sys)
* ffmpeg

## Installation
1. Download or clone gilfoyle-bitcoin-alert.py to your favorite location.

## Usage
Run gilfoyle-bitcoin-alert.py.
